    Alexa Skill - Virtual Traveler

    Purchased an Echo Dot and figured why not try to utilize my studies with AWS Lambda.. Developed an Alexa Skill called 'Virtual Traveler'. Simply ask, "Alexa, ask Virtual Traveler where should I travel to next". Alexa will reply with a hidden gem of a location she suggests you travel to. I utilized Alexa to invoke a simple Lambda function when Virtual Traveler was triggered. I created an array of locations throughout the world that Alexa chooses to share at random upon request. This skill is published/certified and available for download in the Alexa Skills store.

    Optimized Nightly Automation

    Optimized a nightly automation tool to compare 'CVU' specific test cases for our international products. This project began with creating a tool, I used Java, that ran through a file system and then wrote out/created CVU files that contained a trimmed version of the initial file. In some cases, the data changed from 1.5 gb to 500 kb, by using the latest test cases only. Integrated this CVU creation jar into a C# automation program, that allowed a parameter to be passed in that ran and compared the CVU cases instead of the full file system. Another parameter was added to be able to run the data and comparison without calling the jar. Sped up the team's efficiency on certification day by optimizing the cases being ran with our release build.
